Immersive Bag and Display of Phones


The micro project involves creating an interactive and immersive experience featuring a 3D model of a bag and a collection of phone images. Users can explore the 3D model of the bag, and upon interacting with the model, they are presented with various phone images. The interface allows users to view detailed information about each phone and navigate through the images. Key features include a zoom-in effect for the 3D model, image transitions, and interactive elements that enhance user engagement.
